# 纯C++ 捕鱼游戏  
语言:C++  
编辑器:VS2015  
渲染:D2D  
碰撞:obb和aabb    
## 项目文件目录说明
img 放游戏所使用的图片文件夹  
frame 放控制游戏场景相关的文件  
instance 放游戏窗口进程 游戏循环 工具类 碰撞类的文件  
render 放游戏渲染引擎的文件夹  
trigger 放对一些消息进行映射的文件夹
